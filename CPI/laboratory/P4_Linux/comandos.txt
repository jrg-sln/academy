Comandos de la última línea

:w! nombreArchivo	// Guarda y sobreescribe
:wq
:x
:q!

Movimiento del cursor

K	// línea abajo
J	// línea arriba
H	// Carácter a la derecha
L	// Carácter a la izquierda

Retorno de carro	// Salto de línea
w	// avanza una palabra
b	// va al inicio de la palabra actual
e	// va al fin de la palabra actual

?cadena		// busca la cadena hacia atrás
